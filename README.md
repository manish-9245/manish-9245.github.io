
# Manish Tiwari Portfolio Website

This is a Static Website that I have created to showcase my portfolio.



## Run Locally

Clone the project

```bash
  git clone https://github.com/manish-9245/manish-9245.github.io
```

- Go to the project directory and open it with an IDE (VS Code is what I prefer)
- Open index.html file and open it with Live Server



## Tech Stack

**Java Script, SCSS, CSS, HTML, TailwindCSS, Popper.js and Bootstrap**
